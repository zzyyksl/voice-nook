## 给主人的话

这篇更简单，你家机只干一件事：换一个文件。把这个 txt 和随附的 theme-page.html 一起发给你家机，说"照美化篇换上"就行。

## 给机的安装说明（三步）

1. **备份旧页面**（万一想回去）：
```bash
cp ~/.claude/voice/page.html ~/.claude/voice/page-old.html
```

2. **换上新皮**：把随附的 theme-page.html 放到语音目录并改名：
```bash
cp theme-page.html ~/.claude/voice/page.html
```

3. **改名字**：文件里有 3 处 "Rhys" 字样（页面标题、thinking 标题、历史记录里的署名），全局替换成你家机的名字：
```bash
sed -i 's/Rhys/你家机的名字/g' ~/.claude/voice/page.html
```
（"Rhys在想…"那类状态提示文字在 mcp-server.mjs 里，装语音篇时应该已经换过了）

完成。不需要重启任何东西——页面是每次打开现读的，主人把小程序完全关掉重开就能看到。

## 注意事项

- 日间模式 thinking 那块木牌，字是实时渲染的（为了跟着你家机的名字走），字体是系统字体，和卷心菜家原版的手绘字体会有一点点不一样，属正常
- thinking 木牌右侧的小签会显示实时条数；木牌上两个小黑点是画里的钉子，不是脏东西
- 文件比较大（约 700KB，素材全内嵌的缘故），第一次打开慢一两秒正常
- 素材说明：夜间底图为公共领域雕版画（Gustave Doré 风格），日间手绘素材为社区定制生成，仅供个人使用，请勿商用

## 常见问题

**Q: 换完打开还是老样子？**
小程序有缓存，从 Telegram 里彻底关闭小程序再重新打开。

**Q: 我不喜欢日间/夜间的某个颜色能改吗？**
能，让你家机搜 CSS 里对应的色值改，都是明文写着的。

**Q: 只想要其中一套主题？**
默认打开就是夜间，不点切换钮就永远是夜间；反之亦然，让机把默认值 'night' 改成 'day' 即可。
