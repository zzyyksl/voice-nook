> 作者：卷心菜 & Rhys（小红书 5479370809）
> 前提：先装好《语音通话小程序全教程》那套，本篇只是换皮，不改任何功能。
> 已经装过老版美化篇的，看同目录《更新篇.txt》更省事。

效果：
- 夜间模式「黑曜石」：多雷雕版画天使打底，深色玻璃卡片，底部玻璃长条按住说话，银白光线下划线，录音时白色呼吸光圈
- 日间模式「小花园」：奶油纸底，底部藤蔓长条当说话按钮（卷心菜手绘叶蔓），木牌标题（历史记录那块有只黑猫蹲着），水彩小盆栽从卡片边探头，鼠尾草绿气泡
- 录音键在屏幕底部通栏，历史记录再长也不用翻回顶部去按
- 标签栏右上角有个「夜／日」小钮，点一下切换，选择会记住
- 图片素材拆成独立文件走浏览器缓存：页面本体只有 24KB 秒开，图片第二次打开连请求都不发

## 给主人的话

把这个 txt、随附的 theme-page.html 和 assets 文件夹一起发给你家机，说"照美化篇换上"就行。

## 给机的安装说明（五步）

1. **备份旧页面**（万一想回去）：
```bash
cp ~/.claude/voice/page.html ~/.claude/voice/page-old.html
```

2. **换上新皮**：把随附的 theme-page.html 放到语音目录并改名，图片素材放进 assets 子目录：
```bash
cp theme-page.html ~/.claude/voice/page.html
mkdir -p ~/.claude/voice/assets
cp assets/* ~/.claude/voice/assets/
```

3. **改名字**：文件里有 3 处 "Rhys" 字样（页面标题、thinking 标题、历史记录里的署名），全局替换成你家机的名字：
```bash
sed -i 's/Rhys/你家机的名字/g' ~/.claude/voice/page.html
```
（"Rhys在想…"那类状态提示文字在 mcp-server.mjs 里，装语音篇时应该已经换过了）

4. **服务器加图片路由**：新页面的图片走 `/assets/` 路径，需要 mcp-server.mjs 认识它。用本仓库最新版 mcp-server.mjs 的不用动（已内置）；老版本的照《更新篇.txt》第 4 步打两行补丁即可。

5. **重启**：改了 mcp-server.mjs 就重启 claude（服务器跟着 claude 起停）；只换了页面则不用重启，主人把小程序完全关掉重开就能看到。

## 注意事项

- 日间模式 thinking 那块木牌，字是实时渲染的（为了跟着你家机的名字走），字体是系统字体，和卷心菜家原版的手绘字体会有一点点不一样，属正常
- thinking 木牌右侧的小签会显示实时条数；木牌上两个小黑点是画里的钉子，不是脏东西
- 图片缓存 7 天（cache-control: immutable），换图片素材时记得改文件名，不然客户端要等缓存过期才能看到新图
- 素材说明：夜间底图为公共领域雕版画（Gustave Doré 风格），日间藤蔓为卷心菜手绘，其余日间素材为社区定制生成，仅供个人使用，请勿商用

## 常见问题

**Q: 换完打开还是老样子？**
小程序有缓存，从 Telegram 里彻底关闭小程序再重新打开。

**Q: 图片全裂了/显示不出来？**
`/assets/` 路由没生效。检查 mcp-server.mjs 是否加了图片路由（第 4 步），加完要重启 claude；再确认图片确实在 `~/.claude/voice/assets/` 里。

**Q: 我不喜欢日间/夜间的某个颜色能改吗？**
能，让你家机搜 CSS 里对应的色值改，都是明文写着的。

**Q: 只想要其中一套主题？**
默认打开就是夜间，不点切换钮就永远是夜间；反之亦然，让机把默认值 'night' 改成 'day' 即可。
